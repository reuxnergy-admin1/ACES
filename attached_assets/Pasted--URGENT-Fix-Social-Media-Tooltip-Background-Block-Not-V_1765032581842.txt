## LIKELY ROOT CAUSES TO CHECK

### Issue 1: Wrong Element or Structure
The tooltip might be applied to the wrong element:

```jsx
// WRONG - styling on wrapper instead of visible element
<div className="bg-white rounded-lg shadow-xl"> 
  <span className="absolute">WhatsApp</span>  // This has no background
</div>

// CORRECT - styling on the actual tooltip element
<div className="relative">
  <span className="absolute bg-white rounded-lg shadow-xl px-6 py-2">
    WhatsApp
  </span>
</div>
```

### Issue 2: Using a Library Component
Are you using a tooltip library (like Radix UI, Headless UI, React Tooltip)?

```jsx
// If using a library like:
<Tooltip.Root>
  <Tooltip.Content>WhatsApp</Tooltip.Content>
</Tooltip.Root>

// The styling must go on the Content component, not elsewhere
<Tooltip.Content className="bg-white rounded-lg shadow-xl px-6 py-2 border">
  WhatsApp
</Tooltip.Content>
```

### Issue 3: CSS Module or External Stylesheet Override
Check if there's custom CSS removing the background:

```css
/* Check for rules like these in CSS files: */
.tooltip { background: none !important; }
[class*="tooltip"] { background-color: transparent; }
span[class*="absolute"] { background: inherit; }
```

### Issue 4: Pseudo-element Tooltip
Is the tooltip created with `::before` or `::after`?

```jsx
// If using pseudo-elements, the styling must be in CSS, not className:
<button className="has-tooltip">Icon</button>

/* CSS needed: */
.has-tooltip::after {
  content: attr(data-tooltip);
  background: white;
  border-radius: 8px;
  padding: 8px 16px;
  /* etc */
}
```

## Alternative: If Using Tailwind @apply

If you're using CSS with `@apply`, create this in your CSS file:

```css
/* In your CSS file (globals.css or component CSS) */
.social-tooltip {
  @apply absolute left-1/2 -translate-x-1/2 bottom-full mb-2;
  @apply px-4 py-2 text-sm font-medium text-gray-900;
  @apply bg-white rounded-lg shadow-lg border border-gray-200;
  @apply opacity-0 transition-opacity whitespace-nowrap pointer-events-none;
  
  /* Force these in case of conflicts */
  background-color: #ffffff !important;
  border: 1px solid #e5e7eb !important;
  border-radius: 0.5rem !important;
}

.group:hover .social-tooltip {
  @apply opacity-100;
}
```

Then use it:
```jsx
<div className="relative inline-flex group">
  <a href="..." className="...">Icon</a>
  <span className="social-tooltip">WhatsApp</span>
</div>
```
