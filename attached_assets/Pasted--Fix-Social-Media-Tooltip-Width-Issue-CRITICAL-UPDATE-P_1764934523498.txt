```jsx
// CORRECT approaches:

className="w-auto px-6 py-2 whitespace-nowrap"
className="min-w-max px-6 py-2"
```

**D. Font size too large for container:**

### Step 3: Apply this COMPREHENSIVE fix

Replace the tooltip className with this exact combination:
```jsx
className="absolute bg-black text-white text-sm font-medium px-6 py-2 rounded-lg whitespace-nowrap w-auto min-w-fit shadow-lg"
```

**Explanation of each class:**
- `px-6` - Generous horizontal padding (24px)
- `py-2` - Vertical padding (8px)
- `whitespace-nowrap` - Prevents text wrapping
- `w-auto` - Allows width to grow with content
- `min-w-fit` - Ensures minimum width fits content
- `text-sm` - Reasonable text size
- `rounded-lg` - Rounded corners
- `shadow-lg` - Drop shadow for better visibility
