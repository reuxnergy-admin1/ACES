# Fix Social Media Tooltip - Two Issues

## Problem 1: Text NOT Centered Inside Border
The text "WhatsApp" is not centered vertically and/or horizontally inside the bordered box.

## Problem 2: Horizontal Padding Too Small
The border needs MORE horizontal padding (left/right) to comfortably fit the text with proper spacing.

---

## THE COMPLETE FIX

Replace your tooltip code with this:

```jsx
<span 
  className="pointer-events-none absolute left-1/2 -translate-x-1/2 bottom-full mb-3 px-10 py-2.5 text-sm font-medium text-white rounded-lg border-2 border-white whitespace-nowrap opacity-0 group-hover:opacity-100 transition-opacity duration-200 flex items-center justify-center"
>
  WhatsApp
</span>
```

**Key changes:**
1. **`px-10`** - Much wider horizontal padding (40px left/right) for comfortable spacing
2. **`py-2.5`** - Balanced vertical padding (10px top/bottom)
3. **`flex items-center justify-center`** - Centers text both horizontally and vertically inside the box

---

## Alternative: Even More Padding

If you want EXTRA comfortable spacing:

```jsx
<span 
  className="pointer-events-none absolute left-1/2 -translate-x-1/2 bottom-full mb-3 px-12 py-3 text-sm font-medium text-white rounded-lg border-2 border-white whitespace-nowrap opacity-0 group-hover:opacity-100 transition-opacity duration-200 flex items-center justify-center"
>
  WhatsApp
</span>
```

**Changes:**
- **`px-12`** - 48px left/right padding (very comfortable)
- **`py-3`** - 12px top/bottom padding (more vertical space)

---

## If Text is Still Not Centered

Add inline styles to force centering:

```jsx
<span 
  className="pointer-events-none absolute left-1/2 -translate-x-1/2 bottom-full mb-3 text-sm font-medium text-white rounded-lg border-2 border-white whitespace-nowrap opacity-0 group-hover:opacity-100 transition-opacity duration-200"
  style={{
    paddingLeft: '40px',
    paddingRight: '40px',
    paddingTop: '10px',
    paddingBottom: '10px',
    display: 'flex',
    alignItems: 'center',
    justifyContent: 'center',
    textAlign: 'center'
  }}
>
  WhatsApp
</span>
```
